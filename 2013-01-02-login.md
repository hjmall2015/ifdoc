登录．

### Request


```{
    'data': {
	'customerid': '13599990001',	//用户名
	'password': '111111'	//密码
    }
}```

### Response

**如果成功**,　返回如下消息．


```{
    'success': true,
    'data': {
    	'sessionid': '10677352960',
    	'customerid': '13599990001',	//用户ID	
    	'nickname': 'alice',	//昵称
    	'avatar': 'http://company.com/images/NBKAVATAR000000065.jpg',	//头像
    	'balance': 123456	//账户余额
    }
}```

失败响应见 [响应格式及错误码](#/response-status-codes).
