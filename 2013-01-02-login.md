登录．

### Request


```{
    'data': {
	'customerid': 'bob',	//用户名
	'password': '123456'	//密码
    }
}```

### Response

**如果成功**,　返回如下消息．


```{
    'success': true,
    'data': {'sessionid': '10677352960'}
}```

失败响应见 [响应格式及错误码](#/response-status-codes).
